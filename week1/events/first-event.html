    • Designing Intuitive and Mobile Friendly Forms

  • Ask for Required Information Only and Follow a Logical Format
  • Use a Single-Column Format
  • Minimize Typing Effort
  • Field Size and Input Data:Be sure your information can fit comfortably within the given field, not too big, not too small.
  • Format with Flexibility
  • Optional Versus Required Fields:every data input on your form will be required (since you are only asking for necessary information).
  • If you must include an optional input (an example might be a building unit in an address form), be sure to label it as such.
  • Entice Your Users to Fill Out a Form:Provide a list of user benefits for form completion,client testimonials,tell users in explicit terms that you value their privacy
  • Create a Compelling CTA:Let a user know what you do, why they need you, and what sets you apart and do it quickly. Placing your CTA on the main navigation or footer
